O teste a seguir é para testar suas habilidades com Desenvolvimento Front-End Básico. A ideia é testar os seus conhecimentos com HTML, CSS e JavaScript.

Para a conclusão do teste você deve enviar o código em HTML+CSS para um repositório Git público (escolha o que quiser, pode ser GitHub, BitBucket, GitLab ou outro de sua escolha - ele só precisa ser público).

O que você vai fazer:

Transformar essa página em HTML+CSS (com texto e tudo)

  • Onde o ícone da "mãozinha" ( ) aparecer é porque é efeito de hover ou focus;
  • O ícone superior, o "homenzinho" ( ), tem um estado de hover onde o texto fica branco e o círculo fica da cor do background do menu;
  • O menu superior e a caixa de "Favoritos" são fixas;
  • Não use frameworks de CSS (Ex.: Bootstrap)

Aplicar Javascript na página para

  • Ao clicar no ícone do "homenzinho" irá abrir um submenu. Ele tem um status ativo e inativo. Ao clicar precisa trocar o estado (não precisa armazenar em banco de dados, só na seção);
  • O box favoritos ao lado tem que ser filtrado quando digitar um texto no input referente;
  • As estrelinhas devem trocar de estado quando você clicar nos boxes de conteúdo. (Ex.: Grupo Análise Fechamento)

BONUS ROUND (Opcional)

Se você achou tudo muito fácil e terminou o teste, parabéns! Se quiser mostrar um pouco mais o desafio é pegar o código HTML e, em outra pasta do seu projeto, passar o conteúdo para um framework de Javascript da sua escolha (Vue.js, React.js ou Angular). Fique a vontade para escolher.

Lembre-se, isso NÃO é obrigatório, só o teste de HTML e Javascript simples são mandatórios. ;)
